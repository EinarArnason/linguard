Installation
============

As a systemd service
--------------------

1. Download `any release <https://github.com/joseantmazonsb/linguard/releases>`__.
2. Extract it and run the installation script:
    .. code-block:: bash

        chmod +x install.sh
        sudo ./install.sh
3. Run Linguard:
    .. code-block:: bash

        sudo systemctl start linguard.service
